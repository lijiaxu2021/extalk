# ExTalk 管理后台使用说明

## 🎉 全新现代化管理后台

### 访问方式

访问管理后台：`https://comment.upxuu.com/admin`

或者直接访问：`https://fuwari-comments.lijiaxulove.workers.dev/admin`

### 主要功能

#### 1. 登录界面
- ✨ 现代化渐变背景设计
- 🔐 hCaptcha 人机验证保护
- 📧 邮箱 + 密码登录
- 🎨 平滑动画效果

#### 2. 评论管理
- 📊 实时统计卡片（总评论数、页面数、根评论数）
- 🌐 **按 URL 分组筛选** - 可以选择查看特定页面的评论
- 🔍 关键词搜索（支持昵称、内容）
- 👤 用户头像显示
- 🌍 地理位置展示
- 🗑️ 批量删除功能

#### 3. 域名/设置管理
- 🌐 允许的域名列表管理
- ⚙️ IP 属地显示精度（省份/城市）
- 📝 评论最大长度限制
- 📧 邮件同步间隔设置
- 🧪 测试邮件发送功能

#### 4. 用户管理
- 👥 用户列表展示
- 🏷️ 角色标识（管理员/普通用户）
- ✅ 邮箱验证状态
- 📅 注册时间显示

#### 后端 API
- `/admin/comments` - 获取评论列表
- `/admin/domains` - 获取域名列表
- `/admin/users` - 获取用户列表
- `/admin/settings` - 获取/更新设置

### 使用技巧

1. **按 URL 筛选评论**
   - 在评论管理页面，使用"全部页面"下拉框
   - 选择特定 URL 查看该页面的所有评论
   - 结合关键词搜索快速定位

2. **批量操作**
   - 目前支持单条评论删除
   - 未来将支持批量删除、导出等功能

3. **设置管理**
   - IP 属地精度：省份（默认）或城市
   - 评论长度：默认 500 字符
   - 邮件同步：0 表示禁用，>0 表示间隔分钟数

### 注意事项

1. **安全性**
   - 请妥善保管管理员账号密码
   - 定期修改密码
   - 不要在公共场合泄露管理后台地址

2. **性能**
   - 评论数量较多时建议使用 URL 筛选
   - 避免频繁修改设置
   - 测试邮件功能不要滥用

3. **兼容性**
   - 推荐使用最新版 Chrome、Firefox、Safari
   - 不支持 IE 浏览器
